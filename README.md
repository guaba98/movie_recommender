+ 활동 일시: 2023.06.28(수) ~ 2023.07.01(토)
+ 주요 주제: Python, PyQT, DB를 활용하여 영화추천프로그램을 제작한다.
+ 개발 환경: Window OS, PyQt, Qt Desinger, Python 3.11, Pycharm
